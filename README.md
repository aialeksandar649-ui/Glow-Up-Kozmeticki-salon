# Glow Up - Premium Beauty Salon Website 💅✨

A modern, fully animated beauty salon website with advanced UX features built with React, TypeScript, Tailwind CSS, and Framer Motion.

## 🚀 Features

### 1️⃣ Smart Multi-Step Booking Form
- **Step-by-step wizard interface** (NOT a full booking system)
- Animated progress stepper with validation
- 4 intuitive steps:
  - Step 1: Choose service
  - Step 2: Select preferred date
  - Step 3: Pick approximate time
  - Step 4: Enter name & phone
- Beautiful success screen with animations
- Real-time form validation
- Smooth transitions between steps
- **Note**: The form je frontend-only (demo). Podaci se trenutno ne šalju na backend, ali je spremna za povezivanje sa email/booking servisom (npr. Formspree, custom API).

### 2️⃣ Premium Animations & Micro-Interactions
- **Framer Motion** powered animations throughout
- Smooth page transitions
- Hover effects on all interactive elements
- Scroll-reveal animations
- Parallax effects on hero section
- Floating badge animations
- Pulsing CTAs

### 3️⃣ Premium Hero Section
- Large, impactful headline
- Elegant background with animated gradients
- Parallax scroll effects
- Primary CTA: "Zakaži termin"
- Secondary CTA: "Pogledaj usluge"
- Animated floating orbs
- Statistics display (2000+ clients, 5.0 rating, 500+ reviews)

### 4️⃣ Interactive Gallery
- Modal preview with smooth transitions
- Image slider with navigation
- Hover zoom effects
- Keyboard navigation support
- Click to expand images
- Smooth overlay animations
- 9 curated gallery images

### 5️⃣ Modern Pricing Section
- Card-based layout with hover effects
- Filter by category (7 categories)
- Animated entrance on scroll
- 15 different services
- Feature lists with check icons
- Direct booking CTAs
- Special packages section

### 6️⃣ Testimonial Carousel
- Auto-playing slider (5s interval)
- Manual navigation (prev/next buttons)
- Dot indicators for direct navigation
- Pause on hover
- 6 authentic customer reviews
- Star ratings with animations
- Smooth slide transitions

### 7️⃣ Smart Sticky Navigation
- Scroll detection (changes style on scroll)
- Active section highlighting
- Smooth scroll to sections
- Mobile-responsive hamburger menu
- Dark/Light mode toggle in navbar
- 8 navigation links

### 8️⃣ Dark/Light Mode
- Persistent theme storage (localStorage)
- Smooth transitions between modes
- Toggle button with animated icon swap
- All components fully support both modes
- Custom dark mode color palette

### 9️⃣ Additional Premium Features
- **Animated Services Section**: 6 services with hover effects
- **About Section**: Team showcase, statistics, philosophy
- **Contact Section**: Working hours, location map placeholder, social links
- **Premium Footer**: Animated gradient bar, social media icons
- **Custom Scrollbar**: Gold-themed scrollbar design
- **Responsive Design**: Mobile-first approach, works on all devices

## 📝 Customization

### Update Contact Information

Edit the following files:
- `src/components/Contact.tsx` - Contact details
- `src/components/Footer.tsx` - Footer links
- `src/components/SmartBooking.tsx` - WhatsApp/Instagram links

### Update Services & Pricing

- `src/components/Services.tsx` - Service categories
- `src/components/Pricing.tsx` - Pricing items and filters

### Update Gallery Images

- `src/components/InteractiveGallery.tsx` - Gallery items array

### Update Reviews

- `src/components/TestimonialSlider.tsx` - Reviews array
